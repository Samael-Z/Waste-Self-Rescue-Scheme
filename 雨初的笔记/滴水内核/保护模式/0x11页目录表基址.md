### 页目录表基址

+ 拆分4GB？

+ 直接访问CR3看到的是页目录表（**PDT表**），通过 **C0300000** 找到的也是同一个表

  ```
  c0300000
  1100 0000 00
  11 0000 0000 
  
  300 * 4 = C00
  300 * 4 = C00
  0
  ```

### 页表基址

+ `!dd CR3` 可以看到页目录表（PDT表），然后里面的每一个地址对应的都是一张页表（PTT表）。
+ 每一张 **PTT表** 有 **1024个成员，每个成员4字节，加起来4KB大小** 。
