## matplotlib

+ ```python
  import matplotlib.pyplot as plt
  %matplotlib inline
  
  plt.figure() # 相当于分配内存，开启画布。
  plt.plot([1, 0, 9], [4, 5, 6])
  
  plt.show()
  ```

+ `matplotlib` 有三层结构： **容器层** ， **辅助显示层** ， **图像层** 。

  + 容器层
    + 画板层 如果我们使用 `matplotlib` 来作图，其底层已经帮我们创建了一个画板层。

    + 绘图区/坐标系
