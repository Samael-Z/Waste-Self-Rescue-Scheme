案例：计算n!

```
def fact(n):
	s = 1
	for i in range(1, n+1)
		s *= i
	return s
```

> 可以理解为函数就是`IPO`的一种实现，函数也是一段完整代码的封装

### 可选参数传递

计算 n!//m

```
def fact(n, m=1):
	s = 1
	for i in range(1, n+1):
		s *= i
	return s//m
```

```
>>>fact(10)
3628800
>>>fact(10, 5)
725760
```

### 可变参数传递
```
def fact(n, *b):
    s = 1
    for i in range(1, n+1):
        s *= i
    for item in b :
        s *= item
    return s

a = fact(10, 3, 5, 8)
print(a)
```
> 这个就是可以
