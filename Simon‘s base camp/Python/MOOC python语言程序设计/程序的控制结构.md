### 异常处理
```
try: #如果正确则执行try而不执行except
    <语句块1>
except:
    <语句块2>
```    

或者

```
try:
    <>
except <异常类型>:#所有异常名称都是python自定义的
    <>
```

例如：

```
try:
    num = sdfjiae;f 
    print(dsfsd)
except NameError:
    print("输入错误")
```

高级用法

```
try:
    <>
except:
    <>
else
    <>
finally:
    <>
```
